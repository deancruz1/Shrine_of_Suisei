# Shrine of Suisei

A single-page fan site dedicated to Hoshimachi Suisei, the divine comet from Hololive. Built with raw HTML, CSS, and JavaScript as an early front-end project exploring layout fundamentals, custom fonts, and responsive design, done during my time learning under The Odin Project. Inspired by Lyrenide's Hakurei Shrine.

## Features
- **Responsive layout**: Adapts background images, hero section, and navigation for mobile and desktop.
- **Custom fonts**: Six unique `@font-face` declarations including Kawashiro Gothic and Izayoi Monospaced.
- **Glowing CTA button**: Animated gradient border effect on the "Get Enlightened" button using CSS keyframes.
- **Smooth nav animations**: Underline hover effects and letter-spacing transitions on navigation links.
- **CSS-only modal**: Disclaimer popup triggered via `:target` pseudo-class with no JavaScript.
- **Hamburger menu**: Mobile navigation with animated icon toggle and auto-close on link click.
- **Parallax quote section**: Fixed background attachment with dramatic text shadows.
- **External links gallery**: Grid of image cards linking to Pixiv, Reddit, YouTube horror clips, and more.

## Technologies Used
- **HTML**: Semantic markup with accessible structure.
- **CSS**: Flexbox, Grid, custom fonts, animations, `clamp()` fluid typography, responsive media queries.
- **JavaScript**: Hamburger menu toggle, sticky header shadow on scroll.

## Usage
The Shrine of Suisei is a single landing page with four main sections:
- **Hero:** Welcome message, description of Hoshiyomis, and a glowing "Get Enlightened" button linking to Suisei's YouTube.
- **Links:** Image cards linking to Suisei's Pixiv fanart, Reddit community, a horror compilation, and Suicopathic Tendencies.
- **Quote:** A memorable Suisei quote from Project: Winter with dramatic text shadows on a parallax background.
- **Donate:** A call-to-action card encouraging donations with a playful message.

The navigation bar includes links to Suisei's YouTube, Twitter, and official merch shop. A disclaimer modal explains the site's non-commercial purpose.
